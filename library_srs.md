# SOFTWARE REQUIREMENT SPECIFICATION
## DIGITAL LIBRARY PORTAL

### I. INTRODUCTION
- The purpose of this systems is to eneble users to acess a digital library,search books and borrow books

### SCOPE
- Digital library will help users to register have access,search and burrow books

### USERS
- Library holders
- Customers
- Administrators
## II. OVERALL DESCRIPTION

### PRODUCT PERSPECTIVE
- The digital library is a software application use to dowload books

### PRODUCT FUNCTION
The system will
- Create an individual account to each user 
- Give a secure authentification to users
- Log in registered users
- Search for books
- Download books
- Display whether the book is stiil available or not
- borrow books
- Show history of the precendent activities

### USER CHARACTERISTICS
- User should be able to have a computer with them and a minimun ICT skills for using it
---

## III. FUNCTIONAL REQUIREMENT

### Functional Requirement 1: User Authentication
The system shall allow users to register, log in, and log out securely using a username and password.

### Functional Requirement 2: Book Search
The system shall allow users to search for books by title, author, or ISBN and display matching results.

### Functional Requirement 3: Book Borrowing
The system shall allow authenticated users to borrow available books and receive a due date for returning them.

### Functional Requirement 4: Book Return
The system shall allow users to return borrowed books and automatically update the book's availability.
---

## IV NON FUNCTIONAL REQUIREMENT

### SECURITY
-  Users should give personal identification
- The system should be able to secures using a harsh progam user's informations

### PERFORMANCE
- The system should be able to download a book in a time of * 10 seconds*

### USABILITY
- A new student should be able to register in the library within an interval of * 5 minutes*
 
### RELIABILITY
- The system should maintain 99.9% uptime.

---

## V. User Stories

### User Story 1
As a library member, I want to search for books by title, author, or ISBN so that I can quickly find the books I need

### User Story 2
As a library member, I want to borrow available books so that I can read them before the due date
## VI. Acceptance Criteria

For User Story:

- The search bar must suggest books after typing three characters.
- The system must display whether each book is available or unavailable.
- Selecting a book must open a detailed information page.

---
## VI Conclusion

The Digital Library Portal will provide secure, reliable, and efficient access to library services while improving the overall user experience for both members and administrators.
